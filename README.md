- Adjustable:
  - Data size
  - Value range

- Dynamic UI with:
  - Algorithm selection dropdown
  - Input boxes for custom parameters
  - Real-time animations and highlighting
- Built entirely with **Pygame** and **Python** (no external libraries required)
